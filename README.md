# Car Insurance Dashboard

A single-page mockup dashboard for a car insurance funnel.

It shows the main journey from started proposals to completed forms, quoted prices, purchases, and successful payments. The page is built with mock data and is intended as a visual demo, not a production analytics pipeline.

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Checks

```bash
pnpm lint
pnpm build
```
